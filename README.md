# Lucas Daniel de Oliveira Morandi

Sou profissional de mercado financeiro com foco em Renda Variável, automação operacional, BI e dados aplicados a investimentos.

Atuo na criação de soluções que conectam processos financeiros, análise de dados, automação em Excel/VBA, Python, Power BI, Cloud e integrações com APIs.

## Principais frentes

- Automação de rotinas operacionais para mesa de Renda Variável
- Pipelines de dados para relatórios financeiros
- Integração com APIs, webhooks, Cloud Run, GCS e Cloud SQL
- Ferramentas em Excel/VBA para controle, alertas e envio automático de e-mails
- OCR e leitura de documentos financeiros
- Análise de risco, volatilidade e operações estruturadas
- Dashboards e bases para Power BI

## Projetos em destaque

### Cloud Report Ingestion API
Pipeline em Cloud Run para recebimento de relatórios via webhook, armazenamento em GCS, normalização em SQL e disponibilização para Power BI.

### Financial Automation Suite
Conjunto de automações em Python e VBA para envio de e-mails, leitura de planilhas, geração de relatórios e controle operacional.

### PDF OCR Financial Documents
Automação para leitura de PDFs escaneados, identificação de documentos relevantes e estruturação dos dados extraídos.

### Options Risk & Volatility Engine
Modelo para estimar risco de desarme em operações estruturadas com base em volatilidade, barreiras e parâmetros da operação.
